# إستوردلي – Estawredly Store

## 🛍️ متجر إلكتروني متكامل

موقع تجارة إلكترونية احترافي مبني بـ HTML + CSS + JavaScript خالص، يشمل:

- 🏠 **الصفحة الرئيسية** — سلايدر، تصنيفات، منتجات مميزة
- 🛒 **المتجر** — فلترة وبحث وعرض آلاف المنتجات
- 📦 **صفحة المنتج** — معرض صور، تقييمات، اختيار لون/مقاس
- ⚙️ **لوحة الأدمن** — إدارة المنتجات، الطلبيات، العملاء، الإعدادات

## 🚀 تشغيل محلي

```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .
```

ثم افتح: `http://localhost:8080`

## 📁 الملفات

| الملف | الوصف |
|-------|-------|
| `index.html` | الصفحة الرئيسية |
| `shop.html` | صفحة المتجر |
| `product.html` | صفحة تفاصيل المنتج |
| `admin.html` | لوحة تحكم الأدمن |
| `style.css` | نظام التصميم الأساسي |
| `enhancements.css` | تحسينات بصرية إضافية |
| `main.js` | منطق المتجر (سلة، مفضلة، منتجات) |
| `enhancements.js` | تأثيرات تفاعلية (cursor، scroll reveal، ripple) |

## 🎨 التقنيات

- **HTML5** — بنية دلالية كاملة
- **CSS3** — متغيرات، Glassmorphism، animations
- **JavaScript** — Vanilla JS بدون أي framework
- **Google Fonts** — Tajawal + Inter

## 📱 الصفحات المتاحة

- `/` — الرئيسية
- `/shop.html` — المتجر
- `/product.html?id=1` — منتج بالمعرف
- `/admin.html` — لوحة الأدمن
